# 黄大仙灵签

香港黄大仙祠同款灵签，一签一解心诚则灵。装完直接抽：输出协议强制签诗+分类解签完整格式，不跳项不凑字。附Python CLI抽签器内置30+签诗解签数据。

---

## 一句话说明

装上这个Skill，问它抽签或解签，它会按传统格式输出：签号、签题、签诗、典故、整体吉凶、分类详解（事业/姻缘/财运/健康/出行等），一签一解，不跳项。

---

## 装完能干啥

- 直接问：帮我抽支签看看最近的事业运
- 指定签号：我要第25签
- 查看签文详解
- 了解黄大仙灵签100支体系

---

## CLI用法

```bash
# 随机抽签
python huangdaxian.py

# 抽指定签号
python huangdaxian.py 25

# 查看帮助
python huangdaxian.py --help
```
