## 完整示例

### 示例1：随机抽签

**用户输入**：
> "帮我抽一签看看最近的事业运"

**输出**：
```
═══════════════════════════════════
    🏮 黄大仙灵签
═══════════════════════════════════

第32签（下下签）
签题：武则天登位
吉凶：下下

【签诗】
梦中学武则天，
登基亲执白鸾扇；
未央宫内催选，
何日方能再见面。

典故：武则天登位

【解签】
整体吉凶：此签暗示你目前处于权力变动或身份转换的关键时期。虽有雄心壮志，但时机未到，需要耐心等待。

分类详解：
| 事项 | 解读 |
|------|------|
| 事业 | 当前宜守不宜动，勿强求晋升 |
| 姻缘 | 缘分未到，不宜急于求成 |
| 财运 | 财务上需谨慎，避免大额投资 |
| 健康 | 注意休息，不可过度操劳 |
| 出行 | 近期不宜远行 |
| 流年 | 今年运势平平，需耐心积累 |

【典故】
武则天登位：唐高宗驾崩后，武则天以太后身份临朝称制，最终登基为帝，建立武周。此典故暗示权力之路艰难，需有耐心和智慧。

═══════════════════════════════════
```

### 示例2：指定签号查询

**用户输入**：
> "我要第25签"

**处理逻辑**：
- 用户明确指定签号，直接查询该签
- 无需"抽"，直接显示签文和解签

### 示例3：连续抽签处理

**用户输入**：
> "我连抽三签可以吗"

**正确回应**：
```
心诚则灵，一事一签。建议您先想清楚要问的第一件事，抽完再想第二件。
我现在帮您抽第一签：（抽第X签）...
```

**错误回应（需避免）**：
- 连续抽多签而不加引导
- 告诉用户"连抽不灵"

**用户输入**：
> "第X签说的'何日方能再见面'是什么意思"

**输出**：
```
【续解第X签】

"何日方能再见面"在此签中的含义：
- 若问事业：指时机未到，需等待更好的机会
- 若问姻缘：暗示分离中寻找重逢的契机
- 若问出行：行程可能需要推迟

具体还需结合您问事的时间背景来解读。
```
